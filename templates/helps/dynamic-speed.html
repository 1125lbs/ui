The result of equations be used as value for pulsing speed for direct control during printing.

Usable Keywords

[[LayerPosition]]
Absolute layer position in millimeter
[[CurrentPosition]]
Current position in millimeter
[[LayerNumber]]
Current layer number
[[TotalSolidArea]]
Current layer's total solid area in mm^2
[[LargestArea]]
Current layer's largest solid area in mm^2

Math Operations

Math: + - * / **
Calculating different values {5-[[LayerPosition]]*0.1} for each millimeter increase in height of piece decrease cure time for 0.1 second
Conditionals: () < >
Add conditionals to your math logic {([[LayerNumber]]<5)*15+5} if current layer is less than 5, cure time must be 20 seconds not 5 seconds.

By using this feature, any speed setting for direct control including slow section speed will be ignored during printing.

If the value become negative, result will be discarded and normal speed will be used.
